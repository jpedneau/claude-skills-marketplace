# Claude Skills Marketplace

A collection of skills for [Claude Code](https://claude.ai/code) that enhance software engineering workflows.

## Skills

| Skill | Description |
|-------|-------------|
| **write-a-prd** | Create a Product Requirements Document through interactive interview, codebase exploration, and module design |
| **prd-to-plan** | Turn a PRD into a phased implementation plan using tracer-bullet vertical slices |
| **prd-to-issues** | Break a PRD into independently-grabbable GitHub issues |
| **tdd** | Test-driven development with red-green-refactor loop |
| **grill-me** | Stress-test a plan or design through relentless interviewing |
| **write-a-skill** | Create new skills with proper structure and progressive disclosure |
| **design-an-interface** | Generate multiple radically different interface designs for a module using parallel sub-agents |
| **qa** | Interactive QA session where you report bugs conversationally and the agent files GitHub issues |
| **request-refactor-plan** | Create a detailed refactor plan with tiny commits via user interview, filed as a GitHub issue |
| **improve-codebase-architecture** | Explore a codebase to find architectural improvement opportunities, focusing on testability and deep modules |
| **ubiquitous-language** | Extract a DDD-style ubiquitous language glossary from the current conversation |
| **git-guardrails-claude-code** | Set up Claude Code hooks to block dangerous git commands before they execute |

## Installation

Install via the Claude Code skill marketplace, or clone this repo and add it as a local plugin.
